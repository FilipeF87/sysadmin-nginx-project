# Projet Nginx - Serveur Web

## Objectif
Installer et configurer un serveur web Nginx, créer un site web statique et configurer un Virtual Host (Server Block).

## Étapes
1. Installer Nginx
2. Créer le dossier du site
3. Créer un fichier index.html
4. Configurer un server block (virtual host)
5. Activer le site
6. Tester dans le navigateur

## Commandes utilisées

### Installer Nginx
```bash
sudo apt update
sudo apt install nginx -y
